::::::{card} Converter para maiúsculas

Desenvolva uma função que recebe como entrada uma frase e retorna a mesma frase com todas as letras em maiúsculas. Não utilize funções prontas de conversão em sua solução.


:::::{prf:example}
:class: dropdown
:nonumber:

::::{card} Entrada

```
Darlon
```
::::

::::{card} Retorno


```
DARLON
```

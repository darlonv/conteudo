::::::{card} Converter para minúsculas

Desenvolva uma função que recebe como entrada uma frase e retorna a mesma frase com todas as letras em minúsculas. Não utilize funções prontas de conversão em sua solução.


:::::{prf:example}
:class: dropdown
:nonumber:

::::{card} Entrada

```
Darlon
```
::::

::::{card} Saída


```
darlon
```
